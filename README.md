# Peliculasredessociales
Aplicación para gestionar la presencia en redes sociales de una plataforma de películas. Permite buscar publicaciones, analizar el sentimiento de comentarios y programar contenido en redes como Twitter, Instagram y TikTok.

## Funcionalidades
- Buscar publicaciones por palabras clave, hashtags o fechas.
- Analizar el sentimiento (positivo, negativo) de comentarios sobre películas.
- Gestionar datos con operaciones CRUD.
- Programar publicaciones automáticas para anunciar estrenos.

## Instalación
1. Clona el repositorio: `git clone https://github.com/michaeljordan26/Peliculasredessociales.git`
2. Instala dependencias: `npm install` (frontend) y `pip install -r requirements.txt` (backend).
3. Configura las APIs de redes sociales (Twitter, Instagram, etc.).

## Tecnologías
- Frontend: React.js
- Backend: Flask (Python) o Node.js
- Base de datos: MongoDB
- Análisis de sentimiento: NLTK o Hugging Face
